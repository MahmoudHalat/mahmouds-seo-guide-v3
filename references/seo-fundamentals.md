# SEO Fundamentals Reference

## 1. Core Principles That Have Not Changed

- Keyword at the beginning of: page title, URL slug, meta description, H1, first sentence
- Use keyword variations rather than exact-match repetition -- variations correlate far more strongly with rankings than keyword density
- Word count does not correlate with rankings -- length is the outcome of covering a topic, not a target to hit
- Keyword density has ZERO correlation with rankings -- do not count occurrences
- Content created for the searcher outperforms content created for the search engine
- The fundamentals from 15+ years ago still work: authority, relevance, user satisfaction
- Google has not changed what SEO is; they change how they evaluate it for users

## 2. Three Pillars of SEO

### Authority (Third-Party Validation)
- Authority is the #1 reason pages get indexed and rank
- It comes from backlinks, brand searches, click-through rates in top positions
- A link from a page with traffic is worth more than a link from a DA90 homepage with no traffic
- If you have to "index your backlinks," they are worthless -- quality links get indexed naturally
- Authority decays ~85% per link hop: even a billion authority points become near-zero in 3 hops
- Authority is lumpy -- not evenly distributed across a site

### Relevance (Topical Focus)
- Niche down first, expand later -- topical sprawl is penalized in recent updates
- Strong entity definition across a domain matters more than any single page
- Internal linking with contextual anchor text reinforces topical relationships
- Pages that rank have clear topical alignment between slug, title, H1, and content

### User Satisfaction (The Golden Rule)
- Satisfy the searcher's intent completely -- format, depth, and experience
- Google uses click data: good clicks, bad clicks, last longest clicks, pogo-sticking rates
- If 70% of searchers for a query prefer a comparison guide over a product page, Google adjusts rankings accordingly
- UX features matter: optionality (multiple purchase links), visual proof of expertise, price/size selectors

## 3. The Golden Rule

**Satisfy the searcher.** Everything else follows.

- Study the SERP: what format dominates (listicle, guide, product page, video)?
- Check filters/features Google shows: these reveal what searchers actually want
- If Google shows price filters, the intent is commercial with budget consciousness
- If People Also Ask appears, create pages answering those specific questions
- Stop thinking about "what Google wants" -- think about what the person searching actually needs

## 4. Google Algorithm Insights (Leaked Data)

### Four Key Revelations from the 2024 Document Leak

1. **Chrome Data (ChromeInTotal)**: Google uses full clickstream data from Chrome browsers (63% market share) for ranking signals. They denied this for over a decade. Chrome data helps identify real vs bot browsers.

2. **CTR and NavBoost**: Click-through rate IS used in rankings via a system called NavBoost. Google's VP of Search confirmed this in DOJ testimony. Attributes include good clicks, bad clicks, and last-longest clicks. Click farms are less effective because Google validates browser histories.

3. **Site Authority**: A metric literally called "siteAuthority" exists in the leaked documents, found in a quality section (not links). Google denied this while possibly playing semantics about not using third-party domain authority metrics specifically.

4. **Sandbox (HostAge)**: An attribute called "hostAge" is used to sandbox fresh spam at serving time. New sites lacking trust signals are held back. But with sufficient authority signals, sandboxing can be bypassed instantly.

### How Google Knows What Brands Are
- Google connects brands to domains through the domain name itself (nike.com = Nike)
- Google does NOT inherently "love brands" -- it sells your brand name to competitors in ads
- Exact-match domains still provide ranking advantages (not automatic, but significant with authority)

## 5. Common Myths Debunked

| Myth | Reality |
|------|---------|
| Keyword density matters | Zero correlation with rankings |
| Word count targets (e.g., 1,000 words) | Length is an outcome, not a ranking factor |
| XML sitemaps help you get indexed | They help you TRACK indexing rate; they do not cause indexing |
| More crawling = more indexing | Crawling and indexing are separate systems |
| Schema helps you rank | Schema triggers rich snippets IF you already rank; it does not cause ranking |
| LLMs process schema markup | LLMs grab raw text, not structured data |
| FAQ schema gives rich results | Limited to government and health sites since 2023 |
| Domain authority (DA) is a Google metric | It is a third-party metric; Google has a different "siteAuthority" |
| Server-side rendering is mandatory for SEO | Google bots are Chromium-based and can render client-side JS |
| Fresh content always ranks better | If freshness mattered that much, you would constantly be re-editing ranked pages |
| Disavowing links is usually necessary | Google says most sites do not need it; doing it wrong can cause damage |
| Google penalizes for bad hosting neighbors | Partially true for extreme cases, but not as impactful as claimed |
| GEO is fundamentally different from SEO | GEO relies on the same ranking signals; controlling page 1 of Google controls AI citations |
| Links must be from the same industry | Google does not segment why sites link to each other; a plumber can link to a landscaper |
| SEO is dead | The global SEO market was $81B in 2024, projected $171B by 2030, growing ~13%/year |

## 6. SEO Maturity Model (7 Levels)

| Level | Focus | Key Activities |
|-------|-------|---------------|
| 0 -- Unaware | No SEO | No GSC connected, no keyword targeting |
| 1 -- Foundation | Technical basics | GSC setup, sitemap submitted, crawl errors fixed, pages indexing |
| 2 -- Keyword Targeting | On-page SEO | Keyword research, page titles optimized, H1s aligned, URL slugs set |
| 3 -- Content Production | Consistent publishing | Blog cadence, topical clusters forming, internal linking |
| 4 -- Authority Building | Link acquisition | Guest posts, digital PR, partnerships, EMD satellites |
| 5 -- Optimization | Data-driven iteration | CTR gap analysis, content audits, conversion optimization, GSC regex filtering |
| 6 -- Diversification | Multi-channel resilience | YouTube SEO, social signals, email list, community, multiple revenue streams |
| 7 -- Dominance | Market leadership | Topical authority across cluster, AI citation optimization, brand entity control, EMD portfolio |
